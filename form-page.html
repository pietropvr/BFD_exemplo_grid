📝 Formulários: Boas Práticas

  • <label for="id"> sempre associado ao campo
  • Erros visíveis + aria-describedby para leitores de tela
  • Alvos de toque ≥ 44×44px (min-width/height)
  • Ordem de tabulação = ordem visual (Grid auto-flow garante isso)

Boas Práticas em Formulários

  • Associe cada <label> ao seu campo via for e id.
  • Use tipos de input corretos para ativar teclados virtuais adequados.

  • Mantenha a ordem de tabulação idêntica à ordem visual da grade.
